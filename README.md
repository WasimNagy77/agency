# Agency

AI-powered web development agency - from concept to operation.

---

## What This Is

An AI-powered agency that builds websites for clients.

- **AI does most of the work** (Claude Desktop + Claude Code)
- **You oversee and approve**
- **GitHub tracks everything**

## How It Works

```
Client: "Build me a website"
        ↓
Brain (Claude Desktop) creates tasks
        ↓
Workers (Claude Code) execute tasks
        ↓
You review and approve
        ↓
Website delivered
```
